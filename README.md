# My Dotfiles

## Philosophy

This setup is intentionally bare-bones. No status bar, no widgets, no eye candy beyond what's needed. If you're looking for a heavily themed rice, this isn't it — but if you want something light and easy to understand/modify, feel free to use it as a base.


## Dependencies
* Hyprland ( Seperate files for each tweak )
* Conky ( Time and Date )
* Wob ( Volume and Brightness Bars )
* Neovim ( Hmmm )

Optional:
* [HyprDeck](https://github.com/dud-eisme/HyprDeck)
